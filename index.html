How to Protect Your Password

  • ✔ Use at least 12–16 characters for stronger security.
  • ✔ Combine uppercase, lowercase, numbers, and symbols.
  • ✔ Never reuse the same password across multiple websites.
  • ✔ Enable Two-Factor Authentication (2FA) whenever possible.
  • ✔ Use a trusted password manager to store credentials safely.

Password Hazards

Weak passwords can expose you to hacking, phishing, brute-force attacks, credential theft, identity fraud, and financial loss. Avoid names, birthdates, or common words because attackers test these first.
